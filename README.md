#PlayGrade

PlayGrade is a python app that lets users review and rate playgrounds. 

## Features
- Rate Playgrounds across multiple categories
-Saves reviews using SQLite
-Calculates average score
-Displays letter grades (A-F)
-View saved playgrounds in UI

## Tech USed
-Python
-Tkinter
-SQLite
